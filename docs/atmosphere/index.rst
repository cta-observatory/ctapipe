==========================
 Atmosphere (`atmosphere`)
==========================

.. currentmodule:: ctapipe.atmosphere


Introduction
============

This module include all the functions and classes needed for the monitoring
of the atmosphere above the CTA sites.

It consists in three main sub-modules:

* Lidar Calibration

* Fram Calibration

* GDAS Calibration

For more information on where you should implement your code, please have a look to the README.rst files inside each directory.
